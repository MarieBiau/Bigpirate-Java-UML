BIG PIRATE

Pour jouer :
- lancer le fichier .jar en tapant la commande java -jar bigPirate.jar dans un terminal
- sélectionner le nombre de joueurs dans le launcher
- cliquer sur "Lancer la partie"

Jouer le tour d'un moussaillon :
- cocher les cases correspondant aux actions spéciales que vous voulez effectuer 
(décochez et recochez lorsque l'option était déjà sélectionnée)
- cliquez une fois sur "Lancer le dé"
- cliquez sur la case où vous voulez vous déplacer

Jouer le tour du fantôme :
- cliquez sur "Lancer le dé"

Jouer le tour du pirate :
- cliquez sur "Lancer le dé"
- cliquez sur la case où vous voulez vous déplacer
